# Dealerships Fullstack Developer Capstone

This project is a fullstack web application for managing car dealerships, built as part of the Fullstack Developer Capstone. It features a Django backend and a React frontend, with user authentication, dealer and review management, and a modern UI.

## Features
- Django backend with REST API
- React frontend with user-friendly interface
- User registration, login, and authentication
- Dealer listing, filtering, and review system
- Admin panel for management
- Deployed to cloud with CI/CD
